JavaScript, which is abbreviated as JS, is a programming language that conforms to the ECMAScript specification. It was initially created to make web pages alive. JavaScript is lightweight and most commonly used as a part of web pages, whose implementations allow client-side script to interact with the user and make dynamic pages. Using JavaScript, we can change and update both HTML and CSS. 

JavaScript has no link with Java programming language. It is a fully independent language with its specification.
